# Donation Now

## Summary And Purposes

This website is made for providing service to the under privilege people by donating cloths, money, food and other services. In the winter season in Bangladesh there are a lot of people who are suffering in the cold because lack of warm cloths of winter.This is a service providing portal. Through this website we can unite so many people and seeking help and any type of support from them.

### Key Features Uses To Build This Project

- React Router Dom
- Use Context
- Google Auth Provider Authenticated By Firebase
- UseNavigate and Link
- Use state and useEffect hooks
- Javascript collaborated with other library (daisyUI,Animate.css,etc)
